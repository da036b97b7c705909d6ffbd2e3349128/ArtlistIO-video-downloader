Artlist Video Downloader (macOS)

1. Installation
- Move Artlist Downloader.app to your Applications folder.
- Ensure the chromium and ffmpeg folders are in the same directory as the app.

2. How to Run
- Open Finder and find your folder.
- Right-click this folder icon.
- Select New Terminal at Folder.
- Paste "chmod +x start.sh && ./start.sh.
- Paste the Artlist URL, select your resolution, and click Download.

After, you can just double click the file whenever you want to start.
Your browser will open automatically at http://127.0.0.1:8000.

3. File Location
- Videos are saved to: ~/Videos/ArtlistVideos.
